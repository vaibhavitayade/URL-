📌 Project Description

This project is a Flask Web Application that implements a complete User Authentication System.
Users can register, log in, and access a protected dashboard page.

It demonstrates backend development using Flask and database handling using SQLite.

🚀 Features

User Registration (Signup)

User Login

Secure Authentication

Dashboard after Login

SQLite Database Integration

Clean UI with HTML & CSS

🛠 Technologies Used

Python

Flask

SQLite

HTML5

CSS3
